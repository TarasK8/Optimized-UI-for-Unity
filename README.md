# Optimized UI
A package for Unity that is designed to greatly simplify user interface development

## Features
- Most of the built-in UI components have been redesigned to provide more functionality and performance
  - Selectable
  - Button
  - Slider
  - Scrollbar
  - Scrollview
  - Toggle
  - Input Field
  - Dropdown
- New UI components
  - Switcher
  - Progress Bar
